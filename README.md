# 🎵 Spotify Clone (HTML, CSS & JavaScript)

A high-fidelity **Spotify UI Clone** built using **HTML, CSS, and JavaScript**.  
This project recreates the **modern dark-themed Spotify interface** with artist cards, album sections, sidebar navigation, and a music player UI.

---

## ✨ Features

- 🎧 Sidebar navigation (Home, Search, Library)
- 👤 Interactive profile dropdown menu
- 🎤 Popular artists section with circular cards
- 💿 Albums & singles grid layout
- 🎶 Fixed bottom music player UI
- 📱 Responsive layout using Flexbox and Grid

---

## 🛠️ Technologies Used

- **HTML5** – Structure and semantic layout  
- **CSS3** – Styling, Flexbox, Grid, dark theme  
- **JavaScript (Vanilla JS)** – Interactivity and DOM manipulation  
- **Font Awesome** – Icons used in the UI
